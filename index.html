TitanBot

Get StartedInvite

Easy to Use

Titan is designed to be easy to use. All the available commands are listed in the help menu and on this site.

Customizable

There are many ways to customize Titan between server configs, user configs, user profiles, event announcements, and more.

Server Moderation

Titan has multiple server moderation features including blacklisting words that can't be used and muting users in text and voice channels.

User Profiles

Ever user can customize their profile to include various things including their gender, pronouns, height, alignment, and more.

User Relationships

Users are able to create a family tree by asking other users to date, marry, adopt, and more.

Event Announcements

Channels can be subscribed to events such as user joined, user left, and birthdays so that an announcement is sent in that channel when the event is triggered.
